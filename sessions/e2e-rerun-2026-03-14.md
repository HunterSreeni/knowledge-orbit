## Flow 2 -- Newsletter Subscribe (previously FAIL -- 500 error)

**Result: FAIL (still broken)**

Steps reproduced:
1. Scrolled to "Stay in the orbit" section on the post page.
2. Entered `test-rerun@example.com` in the email field.
3. Clicked Subscribe.

**Observed:** Error message "Something went wrong. Please try again." appeared below the form.

**Console error:**
```
Failed to load resource: the server responded with a status of 500 (Server Error)
  @ http://localhost:3001/api/newsletter/subscribe
```

**Root cause:** The server handler at `server/api/newsletter/subscribe.post.ts` calls `serverSupabaseClient` and upserts into a `newsletter_subscribers` table. The 500 indicates either:
- The `newsletter_subscribers` table does not exist in the connected Supabase project, or
- The Supabase connection itself is failing (missing/invalid env vars `SUPABASE_URL` / `SUPABASE_KEY`).

**Screenshot:** `sessions/screenshots/flow2-newsletter-fail.png`
