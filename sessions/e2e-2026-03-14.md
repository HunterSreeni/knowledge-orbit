## Bugs Found

### Bug #1 (Critical) — PostPostShare component not resolving

- **Component:** `PostPostShare` used in `app/pages/posts/[slug].vue` line 241
- **Symptom:** Vue warning: "Failed to resolve component: PostPostShare". The component renders as raw HTML `<postpostshare>` with no interactive elements.
- **Root cause:** Nuxt auto-imports the component at `app/components/post/PostShare.vue` as `PostShare` (not `PostPostShare`). The `.nuxt/components.d.ts` confirms the registered name is `PostShare`.
- **Fix:** In `app/pages/posts/[slug].vue` line 241, change `<PostPostShare>` to `<PostShare>`.
- **Impact:** Share buttons (Twitter/X, LinkedIn, WhatsApp, Copy link) are completely invisible on all post pages.

### Bug #2 (Medium) — Newsletter subscribe returns 500 error

- **Endpoint:** `POST /api/newsletter/subscribe`
- **Symptom:** "Something went wrong. Please try again." shown to user after clicking Subscribe.
- **Root cause:** The `newsletter_subscribers` table likely does not exist in the Supabase database. The API at `server/api/newsletter/subscribe.post.ts` attempts `client.from('newsletter_subscribers').upsert(...)` which fails.
- **Fix:** Create the `newsletter_subscribers` table in Supabase with at minimum an `email` column (text, unique).
- **Reproduction:** Navigate to any post page, enter an email in the newsletter section, click Subscribe.

### Bug #3 (Low) — Author profile shows 0 posts despite having published posts

- **Page:** `/u/huntersreenihspwtest`
- **Symptom:** Stats show "0 posts" but the homepage lists multiple posts by this author.
- **Impact:** Misleading stat on author profile pages.

### Bug #4 (Low) — Hydration mismatch on post content

- **Console warning:** Hydration class mismatch on post content div. Server renders `class="py-10 text-center text-muted text-sm"` (empty state) but client expects `class="prose dark:prose-invert max-w-none"` (content state).
- **Root cause:** The TipTap `generateHTML` function may produce different results on server vs client, or the `content` field evaluation differs during SSR.
